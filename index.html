Redesign of Clozemaster

As part of my EdTech class at Georgia Tech, a group and I chose an application to redesign using the lense of language inequalities. Below I will summarize new and improved features developed to help non-english speaking users have a better experience with the app.

Clozemaster is a freemium language learning app targeted towards users that have had some experience in the language they are trying to learn. The main feature focuses on letting users fill in blanks on culturally contextualized sentences.

Pain Points

Through user research, our group found certain pain points whil using the app.

  • Clozemaster currently contains very limited features to assess how well a user is learning their target language.
  • There are no features that promote user retention through gamification.
  • The app only focuses on flashcard style learning. While this is the main point of the app, diversification of learning styles are important for better knowledge retention.
  • Certain parts of the app are confusing, especially to a user whose native language is not English
  • For an app that boasts incorporation of cultural context, it does not do a good job of doing so.

Proposed Features

Studying the pain points, our group proposed 5 new features to improve Clozemaster through the lense of Language Inequalities

Feature 1: Add audio phonetics to flashcards so users can hear words they are selecting both in their native and learning language
Feature 2: Modify UI text on the application to also include translations from the user's native language.
Feature 3: Add lessons and videos to help a user understand the cultures and traditions of the language they are learning.
Feature 4: Allow users learning the same language from the same original language to speed date and discuss prompted topics.
Feature 5: Incorporate mini-games to make learning more interactive
